Use this method to directly set a page position on screen using x and y coordinates in pixels.


### Syntax

```typescript
RxCore.setPagePosPan(dx, dy);
```

### Parameters

- `dx`: **number** — Position from the left of the page in pixels.
- `dy`: **number** — Position from the top of the page in pixels.

### Returns

- **NA** — This method does not return a value.
