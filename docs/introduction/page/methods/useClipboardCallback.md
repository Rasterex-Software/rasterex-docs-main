This method will re-direct the result of a selected image from [RxCore.getClipRect()](./getClipRect.md) to use a callback instead of sending the image directly to the windows clipboard.

### Associated Callback events.


### Syntax

```typescript
RxCore.useClipboardCallback(onOff)
```

### Parameters

- `onOff`: **boolean** — if true the image data is returned using callback.

### Returns

- **NA** — This method does not return a value.
