Returns a markup object from the currently active document based on its markupnumber property. Be aware that this number may not be unique. 

### Syntax

```typescript
RxCore.getmarkupbyNumber(markupnumber)
```

### Parameters

- `markupnumber`: **number** — property of a markup object.

### Returns

- **markupobject** — A JavaScript object representing the markup. [markup object](../../markup-methods/intro)
