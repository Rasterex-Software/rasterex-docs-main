Turns on/off the markup link tool, enabling the user to add a markup/annotation that has a link associated with it.
When you click on this markup/annotation it will trigger a callback to handle the associated link.

### Callbacks

- [GUI_MarkupLink](../callbacks/GUI_MarkupLink)

### Syntax

```typescript
RxCore.markupLink(selected)
```

### Parameters

- `selected`: **boolean** — Set to `true` to enable the markup line tool, or `false` to disable it.

### Returns

- **NA** — This method does not return a value.
