Use to upload an image to use as a signature element.

### Syntax

```typescript
RxCore.uploadSignature(image, username, binitial)
```

### Parameters

- `username`: **HTML image** - An image object to be saved as a signature.
- `username`: **string** — Unique user name that indentifies a signature image.
- `binitial`: **boolean** — Set to `true` to upload image as initials, instead of full signature.

### Returns

- **NA** — This method does not return a value.

### Associated callback 

- RxCore_GUI_putsignatureComplete - used to handle the returned signature image.
