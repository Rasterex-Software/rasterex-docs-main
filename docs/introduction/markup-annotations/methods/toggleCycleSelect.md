Use this method to turn on or off the automatic cycle selection for markups/annotations.
When on the user can select annotations behind other annotations by clicking multiple times.
When off the user has to change the draw order using [movetoBack](movetoBack) and [movetoFront](movetoFront) in order to select markups/annotations behind other markups/annotations.

### Syntax

```typescript
RxCore.toggleCycleSelect()
```

- **None**
