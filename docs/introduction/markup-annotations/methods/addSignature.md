Activates the drag and drop signature tool.

See developer guide on drag and drop
[Drag and drop](../../Developer-Guide/draganddrop)

Signatures are not listed as an annotation.

### Syntax

```typescript
RxCore.addSignature(selected)
```

### Parameters

- `selected`: **boolean** — Set to `true` to enable the signature tool, or `false` to disable it.



### Returns

- **NA** — This method does not return a value.
