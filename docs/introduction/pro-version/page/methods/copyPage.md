Call this method to copy one or more pages from the currently active PDF document.
A blob representation will be created and accessible to use with [pastePage](./pastePage).

 Use to paste the page into the same or another open PDF document.

### Version
(Pro version only)


### Syntax

```typescript
RxCore.copyPage(pageRange);

```

### Parameters

- `pageRange`: **Double Array of numbers** — A range of pages to copy.


### Returns

- **NA** — This method does not return a value.
