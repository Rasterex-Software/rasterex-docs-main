Extracts a page from a PDF document and open in viewer as a new single page document.

### Version
(Pro version only)

### Syntax

```typescript
    RxCore.openExtractedPageInViewer(fileName, pageNumber);
```


### Parameters
- `fileName`: **string** — Name of the new document.
- `pageNumber`: **number** — The 0 indexed page number to extract.


### Returns

- **NA** — This method does not return a value.
