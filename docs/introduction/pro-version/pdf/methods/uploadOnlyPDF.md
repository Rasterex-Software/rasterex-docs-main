Uploads the currently open PDF document to the back-end without doing an export.

### Version
(Pro version only)

### callbacks
[RxCore.GUI_Upload](../../../file/callbacks/GUI_Upload.md)
    
Requirements.

RxCore version 35.2 or newer.
Latest iframefoxit.js

### Syntax

```typescript
RxCore.uploadOnlyPDF()
```
