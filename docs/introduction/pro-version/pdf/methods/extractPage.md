Extracts one or more pages from a PDF document and download these to a local folder.

### Version
(Pro version only)

### Syntax

```typescript
    RxCore.extractPage(pageRange, remove);
```


### Parameters

- `pageRange`: **double array of numbers[][]** — A double array indicating which pages are to be extracted.
- `remove`: **boolean** — if `true`, the page or pages are removed from the PDF document.

### Returns

- **NA** — This method does not return a value.
