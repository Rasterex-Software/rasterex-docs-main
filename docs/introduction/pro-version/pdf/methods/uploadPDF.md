Use this method to upload the currently modified PDF to the back-end cache folder.

### Version
(Pro version only)

### callbacks
[RxCore.GUI_Upload](../../../file/callbacks/GUI_Upload.md)
    

Requirements.

RxCore version 35.2 or newer.
Latest iframefoxit.js


### Syntax

```typescript
RxCore.uploadPDF()
```

### Parameters

- **NA** — This method does not have any parameters.

### Returns

- **NA** — This method does not return a value.
