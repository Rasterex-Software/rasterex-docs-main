Exports annotations from the loaded PDF to an FDF file in the files cache folder.
This is only for PDF annotations not Rasterex annotations.

### Version
(Pro version only)

### Syntax

```typescript
    RxCore.foxitexportFDF(filetype, toServer);
```

### Parameters

- `filetype`: **number** — A number designating the file type.
- `toServer`: **boolean** — if true (default) the file is posted to the file cache folder on server.


### Returns

- **NA** — This method does not return a value.
