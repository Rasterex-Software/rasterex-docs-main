Use this method to set the magnification for the magnifying tool. [Magnifying glass](magnifyGlass.md)

### Syntax

```typescript
RxCore.setMagnificationScale(bup);
```

### Parameters

- `bup`: **boolean** — Set to `true` to increase the magnfication or `false` to reduce it.

### Returns

- **NA** — This method does not return a value.
