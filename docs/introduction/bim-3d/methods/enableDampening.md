Use this method to turn on/off the inertia effect for 3D orbit mode set by calling [RxCore.orbit3D](orbit3D.md)

(Pro version only. Requires RxCore version 37.43 or newer)

### Syntax

```typescript
RxCore.enableDampening(onoff)
```

### Parameters

- `onoff`: **boolean** — `true` = on, `false` = off.

### Returns

- **NA** — This method does not return a value.
