Use this method to set a custom color for CAD files that is used when switching the CAD background color using [RxCore.toggleBackground](toggleBackground.md)).

### Syntax

```typescript
RxCore.backgroundCustomColor(color)
```

### Parameters

- `color`: **string** — Color string using supported HTML color notation (e.g., `"#00FF00"`).

### Returns

- **NA** — This method does not return a value.
