If the pen table is activated using [RxCore.setPenTable](./setPenTable.md) this method can turn on/off the use of scaling for pen table
rendering. This will affect the visual thickness of the lines depending on zoom factor.


### Syntax

```typescript
RxCore.setPenTableScaled(OnOff);

```

### Parameters

- `OnOff`: **boolean** — `true` = pen table scaling is on, `false` = pen table scaling is off.

### Returns

- **NA** — This method does not return a value.
